Steps to build manual:

1.  Install R package bookdown in R.

2.  Install pandoc and pandoc-citeproc on the OS and make
sure they are on the search PATH for executables.

3.  change directories to: <stan>/src/docs/manual

4.  execute ./_build.sh

5.  The final HTML will be in  <stan>/src/docs/manual/_book/index.html
